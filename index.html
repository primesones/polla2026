Probabilidad estimada de terminar de primero

Simulación Monte Carlo del cuadro restante, con cuotas en vivo de Polymarket — ordenado de mayor a menor probabilidad

¿Por qué no hay una columna "Techo"? Cada planilla tiene un máximo teórico si absolutamente todo le sale perfecto en lo que falta del torneo. Ese máximo suele estar muy por encima de la diferencia de puntos entre el 1° y el último, así que por sí solo casi nunca elimina a nadie — la probabilidad de esta tabla es la que realmente distingue, porque compara escenario por escenario a todas las planillas entre sí, no a cada una por separado.

Sobre "≈0%": significa que esa planilla no ganó ninguna de las repeticiones simuladas — evidencia de una probabilidad muy baja, no una certeza matemática de que sea imposible.
¿Cómo se calcula esta probabilidad? ▾
  • Punto de partida: los puntos ya ganados hoy (partidos jugados), reales, tomados en vivo de Puntuaciones.
  • Cuotas — mercado en vivo de Polymarket: mercado de predicción con dinero real, API pública, sin costo. Para cruces ya definidos se usa la cuota de ese partido puntual; para cruces aún no definidos (semis, 3er puesto, final) se usa la probabilidad real de mercado de cada equipo de llegar a semifinales.
  • Marcador exacto: se calibra un modelo de Poisson propio de cada partido que reproduce exactamente esa cuota, y de ahí se sortea el marcador de cada repetición — no una lista genérica de marcadores igual para todos.
  • Cada repetición es un escenario compartido: se simula un cuadro completo por repetición y se compara a todas las planillas contra ese mismo escenario — no es "el mejor caso de cada quien" por separado (eso es el techo).
  • Bono final (Campeón/Subcampeón/3er puesto): se lee directo de la predicción real guardada en cada planilla.
  • Es una proyección ilustrativa, sensible a las cuotas del momento — útil para ver dónde está la carrera hoy, no una garantía.
